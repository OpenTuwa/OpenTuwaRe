# Manual steps required after deploying these code changes

## Step 1 — Google Search Console

The site is already verified. Go to:
Search Console → Sitemaps → submit both:
- https://opentuwa.com/sitemap-index.xml
- https://opentuwa.com/news-sitemap.xml

## Step 2 — Google News Publisher Center

1. Go to https://publishercenter.google.com
2. Click "Add publication"
3. Publication name: OpenTuwa
4. Publication URL: https://opentuwa.com
5. Add your Google News sitemap URL: https://opentuwa.com/news-sitemap.xml
6. Submit for review (Google manually reviews all news publishers)
7. This review can take 2–4 weeks — submit as soon as the site is live
